# Projet Web Scraping - Introduction

## Description
Petit projet d’initiation au web scraping en Python.  
L’objectif est de récupérer automatiquement des informations depuis un site web (ici [Books to Scrape](https://books.toscrape.com)) pour se familiariser avec les bases du scraping.

## Objectifs 
- Comprendre comment extraire des données d’une page HTML.  
- Utiliser BeautifulSoup pour parser le contenu HTML.  
- Naviguer à travers plusieurs pages d’un site.  
- Créer un DataFrame Pandas pour stocker les données.  

## Prérequis
- Python 3.11 
- Bibliothèques utilisées: pandas, requests, beautifulsoup4
  
## Structure du projet
- Le script parcourt toutes les pages du site et récupère pour chaque livre :  Titre, Prix, Disponibilité, Note
- Les données sont stockées dans un DataFrame Pandas pour permettre une visualisation ou une analyse statistique

# Web Scraping Project - Introduction

## Description
Small introductory project on web scraping in Python.  
The goal is to automatically retrieve information from a website (here [Books to Scrape](https://books.toscrape.com)) to get familiar with basic web scraping concepts.

## Objectives
- Understand how to extract data from an HTML page.  
- Use BeautifulSoup to parse HTML content.  
- Navigate through multiple pages of a website.  
- Create a Pandas DataFrame to store the data.  


## Requirements
- Python 3.11  
- Libraries used: pandas, requests, beautifulsoup4

## Project Structure
- The script goes through all pages of the site and retrieves for each book: Title, Price, Availability, Rating.  
- The data is stored in a Pandas DataFrame to allow visualization or statistical analysis.
